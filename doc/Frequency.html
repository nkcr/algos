    • string2 (String) the second sentence

    # File crypto/frequency.rb, line 29
def friedman(string1,string2)
        j = 0.0
        string1.chars.each_with_index do |char1,i|
                j += ((char1 == string2[i])? 1 : 0)
        end
        j / (string1.length)
end

    calculates Kullback coef of a given string it is done by making friedman of each sentence shifted, divided by the length

    params :

    • string (String) the string

    return :

    • the coef

    # File crypto/frequency.rb, line 45
def kullback(string)
        j = 0.0
        (0..string.length-1).each do |i|
                j += friedman(string,string[i..-1] + string[0..i-1])
        end
        j / string.length
end
